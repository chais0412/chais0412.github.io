App Reserva Fotografías Escolares

Este proyecto fue desarrollado con el propósito de agendar citas para la toma de foto escolares, asi como la selección del paquete o productos.
La aplicación cuenta con la posibilida de realizar una imagen de prueba para entender al usuario de como desea la Fotografía final, esto es
posible ya que se cuenta con un menu interactivo para agregar escudos escolares, nombres, fondos, etc y poder acomodarlos en las posición
que desee el usuario, al finalizar la edición guarda el pedido y le fecah seleccionada para realizar la toma de la fotografía


Las tecnologías que se utilizaron para su desarrollo son las siguientes:

➼ Dart
➼ Flutter
➼ NestJS
